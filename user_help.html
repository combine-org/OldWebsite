- how to customize the site/theme : Administer -> site building -> Blocks. From there, you can control which part is visible, what is inside it and how can see it.

- how to customize the side menu : Administer -> site building -> Blocks. From there, you can configure the existing menus and create new ones.

- how to administer users : Admnister -> User Managment -> users
