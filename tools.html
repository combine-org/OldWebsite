Interoperability tools

Effective interoperability depends on well-crafted standards, but also on the existence of software tools to support them. We gather here information on a few tools developed to facilitate this interoperability. The list is by no mean complete, but will evolve as we get to know other tools. If you think a tool should be listed here, please tell-us at combine-discuss.

Converters

Although COMBINE standards are meant to be orthogonal, it is sometimes possible to generate one core COMBINE representation from another, for instance using additional information coming from the associated standardization efforts

System Biology Format Converter (SBFC)

The System Biology Format Converter is a generic open framework in Java that can potentially translate any system biology format into another. SBFC currently include several converters from SBML, including SBML to BioPAX. An SBML to SBGN-ML converter is under development. An online version is available.

Biological Network Manager (BiNoM)

BiNoM is a software (standalone or used as a Cytoscape plugin) developed to facilitate the manipulation of biological networks represented in standard systems biology formats (SBML, SBGN, BioPAX) and to carry out studies on the network structure. BiNoM provides the user with a complete interface for the analysis of biological networks in Cytoscape environment.

Tools supporting multiple COMBINE standards

One of the goals of COMBINE is to encourage the synergy between the different standard description formats and associated efforts. The following tools help fulfilling this role by integrating information coming from several standards.

Cytoscape

Cytoscape is an open source platform for network analysis and visualization. A very large number of plugins have been designed to extend the capabilities of the software, including several to support COMBINE standards. One can mention CySBML to support SBML, CySBGN to support SBGN, and BiNoM, that supports BioPAX, SBGN and SBML.

The Virtual Cell

The Virtual Cell is a modeling platform that can import and export SBML, and can use BioPAX to build and annotate models. Current versions support querying and downloading pathways in BioPAX from Pathways Commons, a gateway to nine popular pathway databases. Upcoming versions add SBPAX support for quantitative data to build reaction kinetics.

Vanted

VANTED (Visualization and Analysis of Networks containing Experimental Data) makes possible to load and edit graphs, which may represent biological pathways or functional hierarchies. It is possible to map experimental datasets onto the graph elements and visualize time series data or data of different genotypes or environmental conditions in the context of a the underlying biological processes. Vanted supports SBML and BioPAX and the extension SBGN-ED provides full support for SBGN.
